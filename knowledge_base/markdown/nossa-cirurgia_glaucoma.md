# GLAUCOMA

## Como é feita a Cirurgia

As cirurgias para tratar glaucoma podem ser feitas de algumas maneira diferentes, mas todas têm o objetivo de estabilizar a pressão intraocular e evitar a cegueira.  
  
Os procedimentos podem envolver um laser com o objetivo de estimular a drenagem do humor aquoso; uma abertura que funciona como via alternativa para drenagem ou o implante de tubos para o mesmo fim.
