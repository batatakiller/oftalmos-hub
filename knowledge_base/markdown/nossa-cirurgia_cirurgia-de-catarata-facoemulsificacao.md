# Cirurgia de Catarata - Facoemulsificação

## Como é feita a Cirurgia

É uma cirurgia rápida e segura, feita com colírio anestésico. O(A) paciente recebe alta pós cirurgia.  
  
O(A) oftalmologista utiliza uma sonda de ultrassom para fazer micro rupturas na catarata a fim de facilitar sua remoção. Em seguida, é implantada uma lente intraocular (específica para cada paciente) que substituirá o cristalino.
