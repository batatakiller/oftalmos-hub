# ESTRABISMO

## Como é feita a Cirurgia

É um procedimento simples que pode ser feito com anestesia local ou geral.  
  
A depender da necessidade de fortalecer ou enfraquecer os músculos responsáveis pela movimentação dos olhos, o(a) oftalmologista faz pequenos cortes na região do globo ocular para reposicionar esses músculos ou para remoção parcial dos mesmos. Ao final, a conjuntiva é suturada para fechar a incisão. Após algumas semanas, os pontos são absorvidos e não há necessidade de retirá-los.
