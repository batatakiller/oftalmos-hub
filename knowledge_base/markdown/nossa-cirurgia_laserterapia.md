# LASERTERAPIA

## Como é feita a Cirurgia

É um método rápido e objetivo para tratar problemas como catarata, glaucoma, tumores e doenças diabéticas. Ele é bastante indicado por não exigir internação.  
  
Esse método engloba alguns processos similares para tratar esses problemas, sendo eles a fotocoagulação, endolaser, micropulso e indireto.  
  
Eles envolvem o uso de diferentes tipos de laser para queima de tecidos para coagulação, ou para estimular o tecido com o fim de reduzir edemas.  
  
A depender do caso, pode haver a necessidade de uso do colírio anestésico.
