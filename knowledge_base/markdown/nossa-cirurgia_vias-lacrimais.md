# VIAS LACRIMAIS

## Como é feita a Cirurgia

É um procedimento rápido que tem como objetivo abrir uma passagem para as lágrimas escorrerem pelo nariz. A depender da avaliação médica, é feita sob anestesia geral ou sedação leve e realiza-se um dos procedimentos a seguir:  
  
Cirurgia do canal lacrimal: o(a) oftalmologista faz uma incisão no lado do nariz, próximo ao saco lacrimal. Em seguida, coloca uma liga metálica que se expande (stent) criando uma nova passagem entre o sacro lacrimal e a cavidade nasal. Então, a incisão é fechada com alguns pontos.  
  
Sondagem do canal: utiliza-se apenas um estilete que penetra todo o canal com o fim de fazer a desobstrução.
